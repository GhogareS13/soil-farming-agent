How to run the Soil Farming Agent Project

1. Download the  zip file

2. Extract the file and copy soil folder

3. Paste inside root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/html)

4. Open PHPMyAdmin (http://localhost/phpmyadmin)

5. Create a database with name soil

6. Import soil.sql file(given inside the zip package in SQL file folder)

7. Run the script http://localhost/soil (frontend)

8. Run 1.php file
